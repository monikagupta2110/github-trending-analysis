##  Project Overview
This project showcases an interactive Tableau dashboard built to analyze GitHub trending repositories. The analysis focuses on understanding repository popularity, programming language trends, and leading organizations within the GitHub ecosystem. The dashboard helps identify how open-source projects gain traction and how stars and forks are related.

##  Objectives
- Analyze top GitHub repositories by stars
- Identify the most popular programming languages
- Discover leading repository owners and organizations
- Study the relationship between stars and forks
